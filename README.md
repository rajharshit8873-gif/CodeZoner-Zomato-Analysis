# 🍽️ Zomato Restaurant Data Analysis

## 📌 Project Overview
This project analyzes the Zomato Bangalore Restaurants dataset to gain insights into restaurant ratings, pricing, online ordering, table booking, cuisines, and customer preferences. The project also applies Machine Learning models to predict restaurant ratings.

---

## 🎯 Objectives

- Perform Data Cleaning
- Handle Missing Values
- Remove Duplicate Records
- Perform Exploratory Data Analysis (EDA)
- Create Data Visualizations
- Build Machine Learning Models
- Compare Model Performance

---

## 📂 Dataset

- Source: Kaggle
- Dataset: Zomato Bangalore Restaurants Dataset
- Rows: 51,000+
- Columns: 17
